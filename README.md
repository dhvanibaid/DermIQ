# DermIQ – Skincare Intelligence Platform

DermIQ is a skincare analysis project that combines data visualization (Tableau) with a web-based interface to help users understand skincare ingredients and make better decisions.

The project focuses on explaining ingredient safety, skin compatibility, and product effectiveness in a clear and practical way.

## What This Project Includes

This project has two main parts:
- Tableau Dashboard (Data Visualization)
- A dashboard that analyzes skincare data and shows patterns and insights.

 ## Web Application (Frontend + Logic)

A simple interactive platform where users enter their skin details and get personalized results.

Tableau Dashboard
1. Overall Summary
Displays key metrics:
- Total ingredients analyzed
- Average safety score
- Ingredients to avoid
- Best value picks
2. Ingredient Safety Sheet
- Shows safety score for each ingredient
- Helps identify safe vs risky ingredients
3. Skin Type Harshness Heatmap
- Shows how harsh ingredients are for different skin types
- Useful for understanding sensitivity patterns
4. Ingredients by Skin Type
- Shows distribution of ingredients across skin types
- Helps understand which skin type has more relevant ingredients
5. Price vs Efficacy
- Compares product price with effectiveness
- Shows that higher price does not always mean better performance

## Web Application – What It Does

The web app takes user input and generates personalized skincare insights.

1. User Input Form

Users enter:
- Age
- Gender
- Skin type
- Skin concerns
- Sensitivity level
- Budget
- Experience level
2. Personalized Results

Based on input, the system shows:
- Recommended ingredients
- Ingredients to avoid
- Safety scores
- Ingredient benefits
- Harshness level (gentle / moderate / harsh)
3. Additional Insights
- Safer alternatives for harsh ingredients
- Cost-effective product suggestions
- Basic skincare routine (AM/PM)
- Ingredient performance charts
- Tools and Technologies Used
- Tableau for data visualization
- HTML, CSS, JavaScript (React) for frontend
- Chart.js for graphs
- Dataset (CSV/Excel) for ingredient data

## Key Takeaways
- Not all ingredients are suitable for every skin type
- Sensitive skin requires more cautious ingredient selection
- Some commonly used ingredients can be harmful
- Expensive products are not always more effective
- Combining data visualization with user input makes insights more useful

## How to use
- Open the Tableau file
- Use filters like skin type, brand, and price
- Explore charts to understand ingredient safety and trends
- Web Application
- Open the HTML file
- Fill in your skin details
- View personalized ingredient analysis and recommendations
